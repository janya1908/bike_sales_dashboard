# 🚴 Bike Sales Dashboard (Excel)

## Overview

This project showcases an interactive Bike Sales Dashboard created in Microsoft Excel to analyze customer purchasing patterns based on income, age group, commute distance, and demographic filters.

## Tools Used

* Microsoft Excel
* Pivot Tables & Charts
* Slicers

## Key Insights

* Middle-aged customers have the highest bike purchase rate
* Higher income influences purchasing decisions
* Short commute distances show higher purchase trends
* Filters like marital status, region, and education enable dynamic analysis
